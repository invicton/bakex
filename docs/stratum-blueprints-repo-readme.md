# Moved

This document has been superseded. The community blueprint library lives in
[`blueprints/`](../blueprints/) within this repo (not a separate `stratum-community` repo/org),
and its contribution guide is now [`blueprints/CONTRIBUTING.md`](../blueprints/CONTRIBUTING.md).

This file previously described a much larger aspirational scope (CIS L2, STIG, PCI-DSS, HIPAA,
FedRAMP, and ISO 27001 coverage across several OSes) than the library actually contained at the
time, and its CODEOWNERS example referenced a stale identity. Rather than correct all of that in
place, the accurate, current version lives in `blueprints/README.md` and `blueprints/CONTRIBUTING.md`
— see those for the real current inventory, naming convention, and how to contribute.
